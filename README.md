## installation (Termux)

Download Termux:

https://play.google.com/store/apps/details?id=com.termux

Update Termux Packages

```pkg update && pkg upgrade```

```pkg install git```

```pkg install python3```

```git clone```

```cd TeamBinary-DDoS```

```pip3 install -r requirements.txt```

```python3 TeamBinary-DDoS.py```
